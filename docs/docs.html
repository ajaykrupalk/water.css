Documentation

Get started with Water.css, a collection of CSS styles to make simple websites.

Quick Start

Copy paste the link into your head before all other stylesheets

        <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/water.css@2/out/water.css">

Example

        <head>
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/water.css@2/out/water.css">
</head>

Components

A few HTML elements for reference

  • <h1>to <h6> elements
  • <blockquote> element
  • <table> element
  • <form> element
  • <abbr> element
  • <article> element

HTML <blockquote> Tag

The <blockquote> tag specifies a section that is quoted from another source.

            <blockquote cite="https://watercss.kognise.dev/"> 
    Water.css is a drop-in collection of CSS styles to make simple websites like this just a little bit
    nicer.
</blockquote>

Output

Water.css is a drop-in collection of CSS styles to make simple websites like this just a little bit nicer.

HTML <abbr> Tag

The <abbr> tag defines an abbreviation or an acronym.

        <p><abbr title="Cascading Style Sheets">CSS</abbr> is used for styling webpages</p>

Output

CSS is used for styling webpages

A similar output can be got by using the<acronym> Tag
